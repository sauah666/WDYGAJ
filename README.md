# AgentSeeker Skeleton

Архитектурный скелет мульти-режимного агента для поиска работы.
Проект построен на принципах **Clean Architecture** с жестким разделением слоев.

## Структура проекта

*   **core/domain**: Бизнес-логика, сущности, контракты (чистый TS, без зависимостей).
*   **core/usecases**: Сценарии использования, оркестрация потоков данных.
*   **core/ports**: Интерфейсы для взаимодействия с внешним миром (Inversion of Control).
*   **adapters**: Реализация портов (MockBrowser, LocalStorage, MockLLM, ReactPresenter).
*   **presentation**: React-компоненты (View), зависят только от Presenter.

## Запуск

Проект использует ESM-модули и не требует сборщика (Webpack/Vite) для запуска в dev-режиме, если сервер поддерживает ESM.
Для локальной разработки используется `react` через CDN (esm.sh).

## Текущий статус

Реализованы этапы:
1.  **Foundation**: DI, State Management.
2.  **Browser Interface**: Абстракция над браузером.
3.  **Auth & Profile**: Ручной логин, захват профиля, хеширование.
4.  **Targeting**: Генерация стратегии поиска через LLM (Mock).
5.  **Search Config**:
    *   5.1: Сущности поиска.
    *   5.2.1: Логика авто-навигации к странице поиска.

Подробности для разработки см. в `docs/DEV_CONTEXT.md`.
